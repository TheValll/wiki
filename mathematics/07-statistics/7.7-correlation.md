# 7.7 вҖ” Correlation

**What it does:**
A **normalized** version of covariance вҖ” bounded in `[вҲ’1, 1]`, scale-free. Measures the **strength** of a linear relationship regardless of units.

Imagine **temperature measured in В°C vs В°F**. The two measurements have the same relationship (they're just a linear rescaling), but their covariance would be wildly different numerically depending on units. Correlation divides covariance by the standard deviations вҖ” suddenly the numerical answer is the same (`+1`, perfect linear) regardless of units. It answers "how tightly linear is this relationship?" rather than "how big."

**Formula:**
```
               Cov(X, Y)
ПҒ(X, Y)  =  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
             ПғвӮ“ В· Пғбө§

         =  Cov(X, Y) / вҲҡ(Var(X) В· Var(Y))
```

**Interpretation:**
| ПҒ value | Meaning |
|---------|---------|
| `+1` | Perfect positive linear relationship |
| `0.7 to 1` | Strong positive linear |
| `0.3 to 0.7` | Moderate positive |
| `0` | No linear relationship (may still be nonlinearly related!) |
| `вҲ’0.3 to вҲ’0.7` | Moderate negative |
| `вҲ’1` | Perfect negative linear |

**Simple example (same as covariance В§7.6):**
```
X = [1, 2, 3, 4, 5],   ПғвӮ“ = вҲҡ2 вүҲ 1.414
Y = [2, 4, 6, 8, 10],  Пғбө§ = вҲҡ8 вүҲ 2.828
Cov(X, Y) = 4

ПҒ = 4 / (вҲҡ2 В· вҲҡ8) = 4 / 4 = 1.0
```
вҶ’ Perfect linear correlation, as expected for `Y = 2X`.

**Complex example (anomaly detection via correlation breakdown):**
In a healthy electric motor, current draw and rotational speed are strongly correlated (`ПҒ вүҲ 0.95`). If bearings start failing, friction introduces unpredictable load and the correlation **drops** to `ПҒ вүҲ 0.7`. Monitoring correlation over a sliding window is a cheap anomaly detector вҖ” no physics model needed. Used in predictive maintenance for industrial robots.
