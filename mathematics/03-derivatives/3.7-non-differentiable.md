# 3.7 — Non-Differentiable Functions

**What it does:**
Identifies where a derivative **does not exist**. Short section — the goal is just to *recognize* the two common patterns: **corners** and **jumps**.

### Recall — the derivative is a slope

At a smooth point, `f'(a)` is the slope of the **unique tangent line** to the graph at `x = a`. Zoom in on a smooth curve and it looks straight — that slope is `f'(a)`.

A function is **non-differentiable at `a`** when no single slope works at that point.

### Intuition — the car on the road

Drive a car along the graph, steering wheel locked to the curve. The derivative is your heading (compass direction).

| Graph shape | Heading | Differentiable? |
|-------------|---------|-----------------|
| Smooth curve | Well-defined | ✓ |
| Sharp corner | Flips abruptly | ✗ |
| Jump (broken road) | You'd have to teleport | ✗ |

---

### 1) Corner — two different slopes meet

The canonical example uses the **absolute value** function `f(x) = |x|`.

**Reminder — absolute value `|x|`** is the distance from `x` to 0 on the number line. Always ≥ 0.
```
|x| = {  x   if x ≥ 0
      { −x   if x < 0
```
Examples: `|3| = 3`, `|−5| = 5`, `|0| = 0`. It simply "strips the sign".

**Graph of `|x|`** — a "V" made of two straight half-lines meeting at the origin:
```
             y
             │
      \      │      /
       \     │     /
        \    │    /
         \   │   /
     -----•------------  x
             0
             ↑
         corner at x = 0
```

**Slope on each side:**
```
   Left side  (x < 0):  f(x) = −x    →   slope = −1
   Right side (x > 0):  f(x) = +x    →   slope = +1
```

Try to draw "the" tangent at `x = 0`:
```
       \              /
  slope \            /  slope
  = −1   \          /   = +1
          \        /
           \      /
    ────────•────────
           x = 0
```
Which line is the tangent? **Neither** — the two sides disagree. So `f'(0)` doesn't exist.

**Formal check with the limit definition** (`f'(a) = lim_{h→0} [f(a+h) − f(a)] / h`):
```
f'(0) = lim (h → 0)  |h| / h
```
- If `h → 0⁺` (from the right):  `|h|/h = h/h = +1`
- If `h → 0⁻` (from the left):   `|h|/h = (−h)/h = −1`

The two one-sided limits disagree → no overall limit → no derivative at 0.

---

### 2) Jump — the function itself breaks

The canonical example is a **step function** — it jumps from one value to another without passing through anything in between.

```
f(x) = { 0   if x < 0
       { 1   if x ≥ 0
```

**Graph:**
```
             y
             │
      1 ────────────●         ● = closed point (value is reached)
             │
             │                ○ = open point (value NOT reached)
             │
      0 ────○
             │
     ────────┼──────────  x
             0
```

Between the two pieces there is a **vertical gap of height 1** at `x = 0`. The function value "teleports" from 0 to 1 instantly.

**Why no derivative?** A tangent needs a smooth neighborhood to attach to. Here the function isn't even continuous at 0 — there's no nearby behavior to measure a slope against. It's like asking the speed of a car that teleports: undefined.

---

**Rule of thumb:** **not continuous ⇒ not differentiable.**

The reverse is **false** — a function can be perfectly continuous and still fail to be differentiable. The corner `|x|` proves it: no jump at 0, but still no derivative because the two slopes disagree.

**Short example:** `f(x) = |x − 2|` has a corner at `x = 2`. Left slope `−1`, right slope `+1`, so `f'(2)` does not exist.

## Intuition — Non-Differentiable Points: Where the Heading Breaks


A derivative is a **slope**. A slope only exists when a curve has a single, well-defined tangent direction at a point. If the curve does anything weird at that point — a sharp corner, a jump — the slope isn't defined, and the derivative doesn't exist there.

**The physical image.** Imagine driving a car along the graph, with the steering wheel locked to the curve. The **derivative is your heading** — the direction the car is pointing at each moment.

Most of the time, the road is smooth and your heading changes continuously. Two cases break this:

---

### Case 1 — A sharp corner

The road has a kink. You were heading one way; the next instant, you'd have to be heading another way. There's no single direction that works *at the corner itself*.

```
            \                         /
             \                       /
              \                     /
               \                   /
                \                 /
                 \               /
                  \             /
                   \           /
                    \         /
             ────────•────────
                     ↑
               sharp corner
               heading to the left  = one direction
               heading to the right = a different direction
               at the corner itself = both claims, no winner
```

Two sides, two different slopes, no way to pick one. **No derivative at the corner.**

The classic example is `|x|` — a "V" shape with a corner at 0.

---

### Case 2 — A jump

The road is broken. One piece ends, another starts higher up. You'd have to *teleport* over the gap. There's no nearby curve to measure a slope against.

```
                              ●─────────
                              ↑
                        new road starts
                        higher up
           ──────────────○
                         ↑
                   old road stops
```

The function isn't even continuous here. **No derivative at the jump.**

---

### The rule of thumb

> **Not continuous → not differentiable.**

But the reverse is false: a curve can be perfectly continuous and *still* fail to be differentiable. `|x|` is continuous everywhere, but the corner at 0 kills the derivative there.

So the full rule is:

| At a point, the curve is… | Derivative exists? |
|---|---|
| smooth (no corner, no jump) | **yes** |
| has a sharp corner | no — two slopes disagree |
| has a jump | no — not even continuous |

---

### Why care?

Most real-world curves you deal with — trajectories, signals, training losses — are smooth enough to differentiate everywhere. But corner cases matter in practice:

- **Absolute-value loss** in machine learning has a corner at zero error → a subgradient is needed instead of a gradient.
- **Step functions** (on/off signals, indicator functions) are discontinuous → derivative = 0 everywhere except at the jump, where it's undefined (or a Dirac delta in physics).
- **ReLU activation** `max(0, x)` has a corner at 0 → handled with a chosen "subderivative" in backprop.

---

**What to remember.**

A derivative is a **heading** along the curve. If the curve has a **corner**, two headings disagree at that point → no derivative. If the curve has a **jump**, there's no road to ride at all → no derivative. Everywhere else, the heading is well-defined and the derivative lives.
