# 1.8 — Determinant of an Inverse:  det(A⁻¹) = 1/det(A)

**What it does:**
If a transformation scales area by `k`, its inverse scales area by `1/k`. Follows directly from `det(AB) = det(A)·det(B)` applied to `A · A⁻¹ = I`.

Back to the **rubber press**: if the press doubles the area, the inverse press must halve it to return to the original. The inverse's scaling factor is the reciprocal of the forward scaling factor.

## Intuition — The Reciprocal

If A stretches areas by a factor of 5, then `A⁻¹` — the un-machine — must **shrink** them by 5 to put everything back where it was. Scaling by `1/5` is the only factor that undoes a scaling by 5. That is the identity `det(A⁻¹) = 1 / det(A)`, told without algebra.

**The physical image.** You stretch a rubber sheet to 5× its original area. To return it to its original area, you shrink it back by a factor of 5, i.e., you scale by `1/5`. Any other factor would leave the sheet still too big or already too small. The inverse machine's volume factor is forced to be the reciprocal of A's.

---

### Sanity check via §1.7

Running A then `A⁻¹` is the identity machine, whose volume factor is 1 (the identity leaves areas unchanged). By the multiplicative rule from §1.7:
```
det(A) · det(A⁻¹) = det(I) = 1
```
So `det(A⁻¹) = 1 / det(A)`. Algebra and intuition agree because they describe the same fact from two sides.

---

### Signs handle themselves

If A flipped the sheet (`det A < 0`), then `A⁻¹` must flip it back — so `det(A⁻¹)` is also negative (the reciprocal of a negative number is negative). Flip then flip-back = two flips = no net flip, matching `det A · det(A⁻¹) = 1 > 0`. Once again the arithmetic handles what the geometry demands.

**Edge case — why `det A = 0` kills the inverse.** If A destroyed a dimension, its volume factor is 0. A finite reciprocal of 0 does not exist. That is the arithmetic face of "singular matrices have no inverse" — the same fact we met geometrically in §1.5.

**What to remember.** `det(A⁻¹) = 1 / det(A)` is **"to undo a stretch, divide by it"**. The reciprocal is well-defined exactly when A does not crush a dimension (`det A ≠ 0`) — otherwise there is nothing to reciprocate, and the un-machine cannot be built.

## Formula

```
det(A⁻¹) = 1 / det(A)
```

**Proof sketch:**
```
det(A · A⁻¹) = det(I) = 1
det(A) · det(A⁻¹) = 1
⇒ det(A⁻¹) = 1/det(A)
```

## Simple example (2×2)

```
A = [4  7]    det(A) = 10
    [2  6]
A⁻¹ = [ 0.6  -0.7]   det(A⁻¹) = 0.6·0.4 - (-0.7)·(-0.2)
      [-0.2   0.4]              = 0.24 - 0.14 = 0.10 = 1/10 ✓
```

## Complex example (3×3 diagonal)

```
A = [2  0  0]    det(A) = 2·3·4 = 24
    [0  3  0]
    [0  0  4]
A⁻¹ = [1/2   0    0 ]    det(A⁻¹) = (1/2)·(1/3)·(1/4) = 1/24 ✓
      [ 0   1/3   0 ]
      [ 0    0   1/4]
```
