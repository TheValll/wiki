# 1.6 — Determinant

**What it does:**
A scalar summary of a square matrix that tells you (1) whether the matrix is invertible (det ≠ 0), (2) by how much the transformation scales area/volume, and (3) whether it flips orientation (det < 0 means mirror-reflection).

Imagine a **unit square** painted on a rubber sheet. Apply the transformation `A` to the sheet. The determinant is the **signed area** of the resulting parallelogram. det = 2 → square doubled in size. det = 1 → area preserved. det = 0 → the square got squashed into a line (no inverse possible). det < 0 → the square was flipped like a pancake.

## Intuition — The Inflation Factor of a Machine

A matrix reshapes space. The determinant is the **single number that tells you, after the reshaping, how much bigger (or smaller) things became** — and whether space got flipped over like a mirror.

If the unit square has area 1 *before* machine A goes to work, its area *after* the machine becomes `|det A|`. Stretch by 3 in one direction, leave the other alone → area × 3, det = 3. Squash everything flat onto a single line → area 0, det = 0. Flip the sheet over like turning a page → same area, but det becomes **negative** to record the mirror.

That is the whole story. The rest is consequence.

**The physical image — the rubber sheet, weighed this time.** Same sheet as §1.4, same machine A. But instead of asking "where does each arrow go?", we now ask a single quantitative question: **by what factor did the sheet grow or shrink?** The determinant answers it with one number, and also tells you whether the sheet got flipped upside-down along the way.

Two consequences fall out of this image immediately:

- **Sign encodes orientation.** If the machine preserves the handedness of space (counter-clockwise stays counter-clockwise), det > 0. If it mirrors (handedness is reversed), det < 0. The sign is not bookkeeping — it is the machine confessing whether it reversed orientation.
- **Zero means a dimension was lost.** If the machine squashes the 2D square flat onto a line, the resulting figure has area 0 → det = 0. This is not a corner case; it is the algebraic signature of a matrix that **destroys a dimension**. You cannot un-squash a flat pancake back into a square, which is why `det = 0` matches "not invertible" exactly.

---

### Step-by-step construction (2D)

Take a 2×2 matrix whose two columns are `col₁ = (3, 0)` and `col₂ = (1, 2)`.

**Step 1.** Start with the unit square. Area = 1.
```
     ↑ e₂  at (0, 1)
     │
   ┌─┤
   │ │
   └─•───→ e₁  at (1, 0)
```

**Step 2.** Apply machine A: each basis arrow is replaced by its matching column.
- `e₁ = (1, 0)` becomes `col₁ = (3, 0)` — stretched horizontally by 3.
- `e₂ = (0, 1)` becomes `col₂ = (1, 2)` — tilted up and slightly right.
```
            ↗ col₂  at (1, 2)
           ╱
          ╱
         •────────────────────→ col₁  at (3, 0)
```

**Step 3.** The unit square has become a **parallelogram** whose two sides are `col₁` and `col₂`. That parallelogram is the image of the square under the machine.
```
         ╱──────────────────╱
        ╱                  ╱
       ╱                  ╱
      •──────────────────→
            (col₁ along the base, col₂ carrying the tilt)
```

**Step 4.** Read off the determinant.
- **Area of the parallelogram** = width 3 × vertical height 2 = **6**. (The horizontal tilt does not change the area — only the shape.)
- **Orientation check**: going `col₁ → col₂` still rotates counter-clockwise, same handedness as `e₁ → e₂` before the machine. No flip.
- → **det A = +6**. The machine multiplies every area by 6.

---

### The three regimes

**Case 1 — det > 0 : expansion (or shrinking), orientation preserved.**

The parallelogram spans a positive area and the columns turn in the same rotational order as `e₁ → e₂`. The machine scales the sheet by `|det|` and keeps its handedness. This is the "ordinary" regime — rotations, stretches, shears with det > 0.

**Case 2 — det = 0 : a dimension collapsed.**

The two columns are **parallel** — one is a scalar copy of the other. The "parallelogram" has zero height, crushed to a line segment.
```
     •──────→ col₁
     •──────→ col₂   ← parallel to col₁ → square crushed onto one line
```
Area = 0 → det = 0. The machine has irreversibly destroyed a dimension. No rebuilding possible: many different input points all land on the same line, so there is no way to ask "which original point did this come from?". This is the geometric heart of **"singular matrix"** = **"not invertible"** — same fact, two vocabularies.

**Case 3 — det < 0 : the sheet got flipped.**

The columns turn in the **opposite** rotational order: going `col₁ → col₂` is now clockwise, whereas `e₁ → e₂` was counter-clockwise. The sheet has been mirrored.
```
     •──────→ col₁   (pointing right, like e₁)
      ╲
       ╲
        ↘ col₂       (pointing down — the opposite side of e₂)
```
Area magnitude is still `|det|`, but the sign records the flip. Think of flipping a piece of paper over — same area, but left-handed now.

---

### 3D — same story, with volume

In three dimensions, the unit cube becomes a **parallelepiped** (a tilted box) under machine A. Its signed volume **is** det A:

- **det > 0**: right-handed stays right-handed, volume multiplied by `det`.
- **det = 0**: the three columns lie in the same plane (or collapse further) — the cube is squashed flat, volume 0.
- **det < 0**: the orientation has flipped (right-handed → left-handed), volume magnitude is `|det|`.

The determinant generalizes to any dimension as **the signed n-dimensional volume of the parallelotope spanned by the columns**. Same picture, same three regimes, more coordinates.

---

**What this makes obvious.**

- **Why `det(AB) = det(A) · det(B)`.** Two machines in series each apply their own volume factor. Machine B multiplies area by det B, then machine A multiplies the already-scaled area by det A. The combined factor is the product — "stretching twice multiplies the stretches", nothing to memorize.
- **Why `det(A⁻¹) = 1 / det(A)`.** The inverse is the machine that undoes A. To cancel a volume-by-5 stretch you have to shrink by 5 — i.e., multiply by 1/5. Inverses are the reciprocal of the stretching factor.
- **Why `det = 0 ⇔ non-invertible`.** We saw it above from two sides: geometrically (a dimension has been crushed, cannot be rebuilt) and algebraically (no reciprocal of 0). Same fact, two voices.
- **Why parallel columns ⇒ det = 0.** Two parallel arrows span a flat segment, not a real parallelogram. The "recipe" of §1.3 can only reach points on a single line — the seed of non-invertibility we glimpsed there is now measured by a single number: **zero**.

**What to remember.** The determinant is the **signed volume factor** of a matrix, seen as a machine: its **magnitude** = by how much areas (2D) or volumes (nD) get multiplied; its **sign** = whether orientation is preserved (+) or flipped (−); its **zero** = whether a dimension has been destroyed (⇔ non-invertible). Every property of det that looks like a formula reads off directly from this picture — no algebra to memorize.

## Formula

For 2×2:
```
det[a  b] = ad - bc
   [c  d]
```
For 3×3 (Sarrus rule):
```
det[a  b  c]
   [d  e  f]  =  aei + bfg + cdh  -  ceg - bdi - afh
   [g  h  i]
```

## Simple example (2×2)

```
A = [3  8]
    [4  6]
det(A) = 3·6 - 8·4 = 18 - 32 = -14
```
→ Area scaled by 14, orientation flipped.

## Complex example (3×3 via Sarrus)

```
A = [1  2  3]
    [4  5  6]
    [7  8  10]

det(A) = 1·5·10 + 2·6·7 + 3·4·8
       - 3·5·7 - 2·4·10 - 1·6·8
       = 50 + 84 + 96 - 105 - 80 - 48
       = 230 - 233 = -3
```
→ det ≠ 0, so `A` is invertible.
